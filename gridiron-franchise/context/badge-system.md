# Football Franchise Mobile Game - Badge System

## Overview
Badges are special traits that provide gameplay bonuses and visual recognition for player abilities. Badges are earned based on attribute thresholds, performance stats, and achievements.

---

## BADGE TIERS

### Tier Levels
- **Bronze** - Good (70-79 in related attribute)
- **Silver** - Great (80-89 in related attribute)
- **Gold** - Elite (90-94 in related attribute)
- **Hall of Fame** - Legendary (95-99 in related attribute)

## BADGE ACQUISITION

### How Badges Are Earned

**Attribute-Based Badges:**
- Automatically awarded when attribute threshold is met
- Tier upgrades when attribute increases
- Lost if attribute drops below threshold

**Performance-Based Badges:**
- Earned through in-game achievements
- Permanent once earned
- Can upgrade tiers with continued performance

**Special Badges:**
- Unique circumstances (undrafted, comebacks, records)
- Usually permanent
- Cannot be upgraded

## BADGE STACKING

### Stacking Rules
- Universal badges + Position badges = both apply
- Multiple badges of same type = highest tier only
- Complementary badges = effects stack (e.g., Speedster + Deep Threat)

# Football Franchise Mobile Game - GM Skills & Perks System

## Overview
The user plays as the General Manager (GM) of their chosen team. Instead of managing GM attributes, the player unlocks and equips **Skills/Perks** that provide buffs and advantages across different aspects of franchise management.

---

## System Mechanics

### Equipment Slots
- **Starting Slots:** 1 skill equipped
- **Max Slots:** 5 skills equipped (unlockable)
- **Unlock Conditions:** Win championships, complete achievements, reach milestones

### GM Points (Currency)
- **Earned From:**
  - Winning championships (+500 points)
  - Playoff appearances (+100 points)
  - Division titles (+50 points)
  - Completing achievements (varies)
  - Season awards (Coach of Year, Executive of Year)
- **Used For:**
  - Unlocking new skills
  - Upgrading skill tiers

### Skill Tiers
- **Bronze** (100 GM Points) - Basic buffs
- **Silver** (250 GM Points) - Stronger buffs  
- **Gold** (500 GM Points) - Game-changing buffs

## Skill Progression System

### Unlocking Skills
1. Earn GM Points through gameplay
2. Navigate to "GM Skills" menu (in MORE tab or Settings)
3. Browse available skills by category
4. Purchase skill with GM Points
5. Skill is added to your skill library

### Equipping Skills
1. Open "Active Skills" menu
2. Drag & drop skills into available slots
3. Can change equipped skills between seasons (not mid-season)
4. Some skills stack (e.g., multiple Position Guru skills)

### Upgrading Skills
1. Must own Bronze tier to unlock Silver
2. Must own Silver tier to unlock Gold
3. Upgrading costs additional GM Points
4. Can skip tiers by paying cumulative cost (e.g., Bronzeâ†’Gold = 100+250+500)

## Starting Configuration

### New Player (First Season)
- **GM Points:** 0
- **Equipment Slots:** 1
- **Unlocked Skills:** None
- **Tutorial:** Brief explanation of system after first game

### Veteran Player (Multi-Season)
- **GM Points:** Accumulated over career
- **Equipment Slots:** 1-5 (based on achievements)
- **Unlocked Skills:** Based on past purchases
- **Prestiged Skills:** Special gold-tier variants for legendary GMs

## Unlock Conditions for Equipment Slots

### Slot 1 (Default)
- Available from start

### Slot 2
- **Unlock:** Win 1 playoff game OR complete 3 seasons

### Slot 3
- **Unlock:** Win 1 championship OR reach 50 total wins

### Slot 4
- **Unlock:** Win 2 championships OR complete 5 seasons with 10+ wins each

### Slot 5
- **Unlock:** Win 3 championships OR build a dynasty (3 championships in 5 years)

## Balancing Considerations

### Skill Power Levels
- Bronze skills: Noticeable but not game-breaking
- Silver skills: Significant advantage
- Gold skills: Powerful but requires major investment

### Stacking Rules
- Most skills: Only 1 per type can be equipped
- Position Guru: Can stack multiple for different positions
- Some skills have diminishing returns when combined

### Economy Balance
- Average GM Points per season: ~200-300 (with decent performance)
- Championship run: ~500-800 points
- Time to max out skills: ~5-7 successful seasons
- Keep scarcity so choices matter

---

## Future Expansion Ideas

### Prestige Skills (Post-Launch)
- Unlock after winning 5+ championships
- Ultra-powerful gold+ tier skills
- Legendary GM status

### Seasonal Skills (Limited Time)
- Special event skills available for limited seasons
- Unique buffs not in standard pool
- Creates FOMO and engagement

### Skill Sets (Combos)
- Equipping certain skill combinations unlocks bonus effects
- Example: "The Scout" set (3 scouting skills) = extra draft pick
- Example: "The Builder" set (development skills) = faster team chemistry

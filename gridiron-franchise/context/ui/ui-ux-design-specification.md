# Football Franchise Mobile Game - UI/UX Design Specification

## Design Principles
- **Mobile-First:** Optimized for 320px-428px width screens
- **Card-Based Layout:** All content presented in card components
- **Thumb-Friendly:** Bottom navigation, critical actions within thumb reach
- **Dark Theme:** "Sunday Lights Premium" aesthetic
- **Gesture-Based:** Swipe, pull-to-refresh, long-press interactions
- **Quick Access:** Maximum 3 taps to any feature

## Gesture Interactions

### Swipe Gestures
- **Swipe Left on Player Card:** Quick actions (Trade, Release, Bench)
- **Swipe Right on Player Card:** View full profile
- **Swipe Down (Pull-to-Refresh):** Update content on any tab
- **Swipe Left/Right on Tabs:** Navigate between tabs

### Tap Gestures
- **Single Tap:** Select/Open
- **Double Tap:** Quick action (e.g., favorite player)
- **Long Press:** Context menu or detailed info popup

### Other Interactions
- **Pinch to Zoom:** On player cards in roster view
- **Drag & Drop:** Reorder depth chart
